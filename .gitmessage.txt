# 제목
### [gitmoji] <타입> : <제목>

### 제목은 최대 50글자까지 한 줄로 입력                        .

# 본문
### "어떻게"보다 "무엇을", "왜"를 설명

### 본문은 한 줄에 최대 72글자까지 입력, 여러 줄일 경우 -로 구분                              .

# gitmoji 타입 설명
# ✨ :sparkles: feat: 새로운 기능
# 🐛 :bug: fix: 버그 수정
# 🔨 :hammer: refactor: 코드/구조 리팩토링
# 📋 :clipboard: docs: 문서 추가, 수정, 삭제
# 🎨 :art: design: UI 추가, 수정, 삭제
# 🔥 :fire: remove: 코드/파일 삭제
# 💡 :bulb: comment: 주석 추가, 수정, 삭제
# 🪐 :ringed_planet: build: 빌드 시스템 또는 외부 종속성 변경
# 🤍 :white_heart: chore: 기타 변경사항
